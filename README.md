## Projects

### [OrbisOptimizer](https://github.com/GinoK01/orbisoptimizer)
Open research into server-side optimization for game servers. Deterministic heuristics: adaptive scheduling, relevance scoring, feedback loops modeled after TCP congestion control. Hytale is the starting reference. Everything — decisions, tradeoffs, dead ends — is documented publicly as I go.

### [Heuristic Optimizer](https://ginoarzaga.com/en/heuristic-optimizer)
OrbisOptimizer's predecessor, built for Minecraft. Closed source, but the docs and analysis are out there. This is what convinced me the approach was worth doing properly.

### [IT Support Scripts](https://github.com/GinoK01/IT-Supports-Scripts)
PowerShell for support technicians. Automated diagnostics, client-ready reports. Not glamorous — but if you've ever spent 40 minutes pulling logs by hand, you get it.
